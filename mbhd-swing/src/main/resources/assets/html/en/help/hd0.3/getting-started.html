Getting started

MultiBit is designed to be easy to use. It is intended for domestic (or small scale merchant) users who wish to use a desktop computer for their day to day Bitcoin transactions.

This guide will provide you with the first steps you should take after you have installed MultiBit so that you can be confident about how to get things done safely.

Step 1 - Download and install MultiBit

After downloading the installation program, simply double click the icon and follow the on-screen instructions. In most cases just accepting the defaults will be sufficient.

The "How to Install" guide provides more detailed instructions if you need them.

Step 2 - Create a wallet

The first screen you will see is the Welcome Wizard which takes you through creating a wallet step by step.

The "How to create a wallet" guide provides more detailed instructions if you need them.

Step 3 - Request some bitcoin

After you have created your wallet you can unlock it using your password. This will remove the wizard screens to reveal the main wallet. Selecting the "Send/Request" link on the sidebar will show two buttons. Select the "Request Bitcoin" button to show the request bitcoin wizard.

The "How to request bitcoin" guide provides more detailed instructions if you need them.

Step 4 - Send some bitcoin

In the "Send/Request" screen select the "Send Bitcoin" button to show the send bitcoin wizard.

The "How to send bitcoin" guide provides more detailed instructions if you need them.
