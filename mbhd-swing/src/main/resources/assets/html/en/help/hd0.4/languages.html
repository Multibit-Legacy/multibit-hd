Languages

To choose which language you would prefer MultiBit to use select "Languages" from the Preferences screen

  1. From the "Preferences" screen select "Languages".
  2. In the "Languages" screen select your preferred language and then "Apply".
  3. From then on, menu items (buttons and messages) will then appear in your selected language (where a translation exists).
  4. You can always change your preference by selecting a different language.
