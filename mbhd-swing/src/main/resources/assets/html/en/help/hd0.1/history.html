History

To see the non-financial usage history of your wallet (e.g. when it was opened), select "History" from the sidebar.

  1. You can add a private note to any event, by selecting the event and then selecting "Edit".
  2. To edit the private note of several events at once, select the events, and then select "Edit".
  3. To edit the private note of all events in your wallet, select "All" from the "All/none" dropdown and then select "Edit".

Some additional information

The private notes are stored in your wallet.
