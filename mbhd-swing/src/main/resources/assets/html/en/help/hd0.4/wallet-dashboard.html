Wallet dashboard

Should you need to find out more about the technical aspects of the currently unlocked wallet you can use the dashboard view.

From the Manage Wallet screen select "Wallet dashboard" which will provide the following information:

  • Notes - your wallet description
  • Cloud backup location - normally SpiderOak or Dropbox but can be any synchronized location
  • Application directory - where to look for your personal files (this may be a hidden directory)
  • Wallet directory - the unique name of the directory where MultiBit stores this wallet
  • Contacts - the number of contacts present in this wallet
  • Payments - the number of payments (either sends or requests) in the wallet
  • Wallet capabilities - a technical description of the capabilities of this wallet (e.g. KeepKey compatible)
