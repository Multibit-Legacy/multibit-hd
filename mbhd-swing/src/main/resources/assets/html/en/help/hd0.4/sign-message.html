Sign message

To prove that you own a Bitcoin address, send a signed message together with the address. Anyone can then use MultiBit to verify that the message was signed with the address's private key. From the Tools select "Sign message"

  1. Select an address to sign the message in the "Bitcoin address" field. This must be from the wallet and can be obtained by requesting a payment.
  2. Enter the message to sign.
  3. Enter the wallet password in the "Password" field to allow access to the private key.
  4. Press the "Sign Message" button.
  5. You will see the signature appear below.

The "Clear All" button will erase the contents of all fields.

Bitcoin signed messages

If you select "Copy All" a Bitcoin signed message will be copied to your clipboard. You can then paste this into an email or chat window to transfer the signed message to a recipient. Below is an example.

-----BEGIN BITCOIN SIGNED MESSAGE-----
Hello World
-----BEGIN BITCOIN SIGNATURE-----
Version: MultiBit HD (0.1)
Comment: https://multibit.org
Address: 16R2kAxaUNM4xj6ykKbxEugpJdYyJzTP13

H0b22gIQIfutUzm7Z9qchdfhUtaO52alhNPK3emrkGOfbOzGHVPuWD9rMIphxniwBNgF/YN4c5C/dMwXz3yJz5k=
-----END BITCOIN SIGNATURE-----
